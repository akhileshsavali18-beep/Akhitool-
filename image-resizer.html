Frequently Asked Questions

Will compressing dimensions reduce my picture clarity?

Scaling images downward maintains clarity exceptionally well as pixel matrices are compressed cleanly. However, upscaling micro-resolution files excessively past native dimensions introduces pixelation due to automated system scaling constraints.

Is there a cap on file size limits or concurrent batch jobs?

Because allocations leverage native local device processing overhead rather than cloud servers, there are zero strict processing limits. File limitations rely purely on your device's browser memory headroom thresholds.
